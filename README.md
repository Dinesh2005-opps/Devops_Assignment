# Hope Bridge - Orphanage Donation Platform

## Project Overview

Hope Bridge is a web application that connects orphanages with people who want to donate.
Orphanages can post their needs such as food, clothes, books, or financial help. Donors can view these requests and provide support.

The platform aims to make donations more transparent and accessible.

---

## Features

### User Authentication

* Secure login and registration system
* Role-based access for donors and orphanages

### Orphanage Dashboard

* Post donation requirements
* Add item name, quantity, and description
* Manage posted needs

### Donor Dashboard

* View all orphanage requests
* Choose items or funds to donate
* Support orphanages easily

### Responsive User Interface

* Modern design
* Mobile-friendly layout
* Clean dashboard structure
